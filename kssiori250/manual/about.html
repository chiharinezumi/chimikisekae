きせかえシステム"siori" について

はじめに

きせかえシステム"siori" は、Webブラウザで遊べる着せ替えゲームを作成するソフトです。 服をマウスでドラッグして動かすタイプの着せ替えゲームを作成することができます。 Internet Explorer 7.0以降 / Mozilla Firefox などの JavaScript に対応するブラウザで動作します。

特徴

  • 画像のグループ化やセット機能・スナップ機能などをサポートしているので、本格的な着せ替えを作ることができます。
  • HTML5 Canvasなどの機能に対応しました。画像の透明部分を正確に判定できるようになり、移動の操作がしやすくなりました。
  • 着せ替え作成ツールが用意されていますので、簡単に作成することができます。
  • 特定のブラウザに強く依存しませんので、多くのブラウザで着せ替えを楽しめます。

ファイルの内容

圧縮ファイルの中には、下記のファイルが含まれています。

siori.js
きせかえシステム"siori" 本体スクリプトファイルです。
index.html
着せ替え作成用のテンプレートファイルです。 しおりのどうぐばこで出力されるデータを、このファイルに貼り付けると着せ替えが完成します。
tool.html
着せ替え作成ツール「しおりのどうぐばこ」です。
manual
このマニュアルが含まれているフォルダです。中身は全てマニュアル関係のファイルです。
